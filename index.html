SUBTLE (spectrogram-UMAP-based temporal-link embedding) is an unsupervised animal behavior clustering algorithm (github link).
You can use any types of pose estimation data (AVATAR, Deeplabcut, SLEAP, etc..) to get unsupervised behavior clusters and their analysis results.

SUBTLE is now available in bioRxiv!
